# 📓 MkrProjects_RealMadrid
**Arduino MKR 1010 & IoT Carrier Projects**

This repository contains a collection of **IoT projects** developed using the **Arduino MKR WiFi 1010** board and the **Arduino IoT Carrier**, created collaboratively by a team of students.

## 📖 Overview
The goal of this repository is to design and implement **Internet of Things (IoT)** applications by taking advantage of the sensors and features provided by the Arduino IoT Carrier.

These projects focus on:
- Environmental data monitoring
- Sensor data acquisition
- Wireless communication
- Embedded systems programming

## 🛠️ Hardware
- **Arduino MKR WiFi 1010**
- **Arduino IoT Carrier**
- USB cable
- Optional external components (LEDs, resistors, etc.)

## 💻 Software & Libraries
- **Arduino IDE**
- Required libraries:
  - `Arduino_MKRIoTCarrier`
  - `WiFiNINA`
  - `ArduinoIoTCloud` (if cloud services are used)
